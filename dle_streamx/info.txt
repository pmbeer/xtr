Название: Wink.ru Style Template
Версия: 1.0
Автор: AI Assistant
Описание: Современный шаблон для DLE 16.0 в стиле кинопортала Wink.ru
Дата: 2024
Лицензия: MIT

Особенности:
- Темная тема с оранжевыми акцентами
- Адаптивный дизайн для всех устройств
- Современный UI/UX в стиле Wink.ru
- Поддержка всех основных функций DLE
- Оптимизирован для кинопорталов
- Встроенные анимации и эффекты
- Поддержка Font Awesome иконок
- Кастомные скроллбары
- Cookie consent уведомления
- Функции поиска и фильтрации
- Переключение видов отображения (сетка/список)
- Сортировка контента
- Социальные функции (поделиться, избранное)

Поддерживаемые страницы:
- Главная страница
- Страницы фильмов/сериалов
- Категории
- Поиск
- Комментарии

Требования:
- DLE 16.0+
- Поддержка CSS Grid и Flexbox
- Современный браузер

Установка:
1. Скопировать папку dle_streamx в /templates/
2. Активировать шаблон в админ-панели DLE
3. Настроить категории и контент

Поддержка:
- Адаптивность для мобильных устройств
- SEO оптимизация
- Быстрая загрузка
- Совместимость с современными браузерами
