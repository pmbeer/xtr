## 🛠 Установка

### 1. Подготовка
1. Убедитесь, что у вас установлен **DLE 16.0 или выше**
2. Сделайте резервную копию текущего шаблона
3. Скачайте и распакуйте архив с шаблоном

### 2. Загрузка файлов
1. Загрузите папку `dle_streamx` в директорию `/templates/` вашего сайта
2. Установите права доступа 755 для папки и 644 для файлов

### 3. Активация шаблона
1. Войдите в админ-панель DLE
2. Перейдите в **Настройки** → **Шаблоны**
3. Выберите шаблон **"Wink.ru Style Template"**
4. Нажмите **"Сохранить"**

### 4. Настройка категорий
1. Создайте категории для фильмов, сериалов и других типов контента
2. Настройте права доступа к категориям
3. Добавьте описания для категорий (отображаются в шапке страницы категории)

## ⚙️ Настройка

### Цветовая схема
Основные цвета можно изменить в файле `style.css`:

```css
:root {
    --primary-color: #ff6b35;      /* Основной оранжевый */
    --secondary-color: #ff8c42;    /* Вторичный оранжевый */
    --background-dark: #0a0a0a;    /* Темный фон */
    --background-light: #1a1a1a;   /* Светлый фон */
    --text-primary: #ffffff;       /* Основной текст */
    --text-secondary: rgba(255, 255, 255, 0.8); /* Вторичный текст */
}
```

### Логотип и брендинг
Измените логотип в файле `main.tpl`:

```html
<div class="logo">:: WINK</div>
```

### Промо-баннер
Настройте текст промо-баннера в `main.tpl`:

```html
<div class="promo-text">
    Бесплатный просмотр топовых фильмов и сериалов для всех — не пропустите!
</div>
```

## 🐛 Решение проблем

### Шаблон не отображается
1. Проверьте права доступа к файлам (755 для папок, 644 для файлов)
2. Убедитесь, что шаблон активирован в админ-панели
3. Проверьте консоль браузера на наличие ошибок

### Стили не загружаются
1. Проверьте пути к CSS файлам
2. Убедитесь, что файлы загружены на сервер
3. Проверьте права доступа к файлам

### JavaScript не работает
1. Проверьте консоль браузера на ошибки
2. Убедитесь, что файл `script.js` загружен
3. Проверьте совместимость браузера
