## Wink.ru — Wireframes (MVP set)

This folder contains low/medium fidelity wireframes for the MVP screens across Desktop, Mobile, and Smart TV. Use these as structure references for UI design and implementation planning in DLE 16.0.

### Scope in this drop
- Home
- Content Card (film/series)
- Player (VOD/Series)
- Paywall/Subscription

Each screen file includes:
- Breakpoints: Desktop, Mobile, TV
- Layout grid and key components
- Primary states (loading/empty/error/access/paywall)
- Interaction notes (hover/focus/gestures/D‑pad)

### File map
- `wireframes/home.md`
- `wireframes/card.md`
- `wireframes/player.md`
- `wireframes/paywall.md`

### Conventions
- Base spacing unit: 8px
- Card aspect: 2:3, radius 8px
- Safe area (TV): 5–7%
- Focus visible on TV: 4px outline within safe area
